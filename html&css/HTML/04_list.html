Списки

Для простоты создания списка пишем ul>li{element $}*x где:

  • x это количество элементов списка
  • element это название каждого элемента
  • $ это порядковый номер каждого элемента

В результате этой команды получаем:

  • element 1
  • element 2
  • element 3
  • element 4
  • element 5
или...
  1. element 1
  2. element 2
  3. element 3
  4. element 4
  5. element 5
